### Hands on With Fastai 

For our hands-on session, please choose a problem you're interested in below, and walk through and experiment with the linked example from fastai. These are some really terrific starting points for lots of interesting problems, and you'll get to see the common patterns used in fastai accross various applications. 

1. [NLP - classifying movie reviews](https://github.com/fastai/course-v3/blob/master/nbs/dl1/lesson3-imdb.ipynb)
2. [Tabular - Grocery Store Prediction Problem from Kaggle](https://github.com/fastai/course-v3/blob/master/nbs/dl1/lesson6-rossmann.ipynb)
3. [Multilabel Image Classification](https://github.com/fastai/course-v3/blob/master/nbs/dl1/lesson3-planet.ipynb)
4. [Head Pose](https://github.com/fastai/course-v3/blob/master/nbs/dl1/lesson3-head-pose.ipynb)
5. [Image Segmentation](https://github.com/fastai/course-v3/blob/master/nbs/dl1/lesson3-camvid.ipynb)
